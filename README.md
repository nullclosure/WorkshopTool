# Workshop Tool

A fast, modern Steam Workshop browser for Windows.

## Features

- Search and browse any game's Steam Workshop
- Faster than Steam's native browser
- Subscribe/unsubscribe to mods directly
- View your subscribed mods library
- Create custom collections
- Clean, dark interface

## Download

Get the latest version from [Releases](https://github.com/nullclosure/WorkshopTool/releases).

## Requirements

- Windows 10/11
- Steam installed
- [Steam Web API Key](https://steamcommunity.com/dev/apikey)
